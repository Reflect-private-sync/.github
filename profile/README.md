# Reflect — Fast Encrypted Notes with Private Sync for Privacy‑Conscious Users

## About Reflect

Reflect provides fast encrypted notes with private sync, end‑to‑end zero‑knowledge security for privacy‑conscious users, professionals, and writers. Reflect combines daily notes, backlinks, and Markdown editor. Reflect supports automatic sync, network graph, and linked mentions. Reflect integrates with Readwise, native iOS/macOS features, and global keyboard shortcut. Reflect enables users to capture private ideas, journal daily, and build an encrypted personal knowledge base. Reflect is trusted by privacy advocates, executives, researchers, and note‑taking purists worldwide.
